# 并发请求

默认情况下，最新请求结果将会覆盖上一次的请求结果。这在一些列表上执行更新或者删除时，可能会导致状态被覆盖。

在下面的例子中，你可以尝试点击某个用户的按钮，此时你会发现所有按钮的状态都改变了，这是不合理的。

import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

function userStatus(userId: number) {
  return new Promise<string>(resolve => {
    setTimeout(() => {
      const name = userList.find(i => i.id === userId)?.name;
      resolve(`${name} status update successfully`);
    }, 1000);
  });
}

const userList = [
  { id: 1, name: 'Benny' },
  { id: 2, name: 'John' },
  { id: 3, name: 'Sam' },
];
export default defineComponent({
  name: 'App',
  setup() {
    const { run, loading } = useRequest(userStatus, {
      manual: true,
      onSuccess: data => {
        alert(data);
      },
    });
    return () => (
      <ul>
        {userList.map(user => (
          <li>
            <span>{user.name} </span>
            <button onClick={() => run(user.id)}>{loading.value ? 'loading...' : 'disable'}</button>
          </li>
        ))}
      </ul>
    );
  },
});

因此，当遇到上述类似的问题时，你可以设置 queryKey() 来单独管理每一项请求的状态，然后通过 queries 来访问每一项的数据。

queryKey() 会将请求函数的参数作为入参，你只需要返回一个字符串（确保该字符串在并发列表里是唯一的）给我们来维护并发请求。

import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

function userStatus(userId: number) {
  return new Promise<string>(resolve => {
    setTimeout(() => {
      const name = userList.find(i => i.id === userId)?.name;
      resolve(`${name} status update successfully`);
    }, 1000);
  });
}

const userList = [
  { id: 1, name: 'Benny' },
  { id: 2, name: 'John' },
  { id: 3, name: 'Sam' },
];
export default defineComponent({
  name: 'App',
  setup() {
    const { run, queries } = useRequest(userStatus, {
      manual: true,
      queryKey: id => String(id),
      onSuccess: data => {
        alert(data);
      },
    });
    return () => (
      <ul>
        {userList.map(user => (
          <li>
            <span>{user.name} </span>
            <button onClick={() => run(user.id)}>
              {queries[user.id]?.loading ? 'loading...' : 'disable'}
            </button>
          </li>
        ))}
      </ul>
    );
  },
});
